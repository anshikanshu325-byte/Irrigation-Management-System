# Irrigation Management System
This is the beginner - level C project created in the first semester.
We developed an irrigation management system using structure in C.
The project helps manage water distribution efficiently using simple logic.

## How to Use 
Step 1 : Compile the program using gcc main .c irrigation.c -o irrigation
Step 2 : Run the executable file
./irrigation (Linux)
irrigation.exe (Windows)
